# ShopSmart AI - E-commerce Platform

Welcome to the ShopSmart AI E-commerce repository! This is a modern, AI-powered e-commerce platform built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- **AI Chatbot**: Built-in intelligent assistant that helps customers find products and answers questions automatically.
- **Admin Dashboard**: Comprehensive dashboard for managing products, orders, users, and reviews.
- **Coupon Management**: Admins can generate custom discount codes with percentage or flat rates.
- **User Profiles**: Secure authentication, order tracking, and wishlists for customers.
- **Responsive UI**: A gorgeous, mobile-friendly interface built with React and custom styling.

## Technologies Used
- **Frontend**: React.js, React Router, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **AI Integration**: Groq API (Llama 3 Model)

## Getting Started

### Prerequisites
Make sure you have Node.js and MongoDB installed on your system.

### Running the Project
1. Open a terminal and navigate to the `server` folder. Run `npm install` and then `npm run dev` to start the backend.
2. Open another terminal and navigate to the `client` folder. Run `npm install` and then `npm start` to launch the frontend.
3. Open `http://localhost:3000` in your browser.
